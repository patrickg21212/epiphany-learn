# Module 2: Talking to AI

5 lessons, ~35 min total, 250 XP.

## Files
- `index.json` — Module metadata, lesson list, XP/badge config
- `lesson-1-first-conversation.mdx` — Your First Real Conversation with AI (6 min, 50 XP)
- `lesson-2-better-questions.mdx` — How to Ask Better Questions (7 min, 50 XP)
- `lesson-3-when-ai-gets-it-wrong.mdx` — When AI Gets It Wrong (And How to Spot It) (7 min, 50 XP)
- `lesson-4-ai-for-everyday-tasks.mdx` — AI for Everyday Tasks (8 min, 50 XP)
- `lesson-5-ai-cheat-sheet.mdx` — The AI Cheat Sheet (7 min, 50 XP)

## Target Audience
Everyday people — zero technical knowledge assumed. Writing is conversational, like explaining over coffee.

## Interactive Components Used
- `<Quiz>` — multiple choice questions with explanations
- `<RevealCard>` — click-to-reveal cards for tips and prompts
- `<MatchTerms>` — match tasks to the right tool (AI, Google, human)
- `<LessonComplete>` — end of each lesson
- `<ModuleComplete>` — end of module 2

## Writing Style
- Short paragraphs (2-3 sentences max)
- Everyday analogies (cooking, texting, contractors, friends)
- No jargon without immediate plain-English explanation
- Humor and personality — this should feel fun to read
